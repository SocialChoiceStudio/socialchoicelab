Compound C API — interactive demos

Frames precomputed with socialchoicelab (__SOURCE_TAG__). Drag sliders to scrub. Geometry comes from ic_interval_1d, scs_winset_2d_export_boundary, uncovered_set_boundary_2d (Python uses heap export; R uses the standard .Call path), and scs_voronoi_cells_2d_heap (R: same via r_scs_voronoi_cells_2d).

1 · ic_interval_1d

Compound 1D indifference interval (distance → utility → level set in one C call). Voter ideal fixed at 0; move the reference (e.g. seat).

2 · scs_winset_2d_export_boundary

Winset boundary sampled without creating a Winset handle. Status quo orbits the voter triangle; filled region uses even-odd rule for holes.

3 · uncovered_set_boundary_2d

Approximate uncovered-set boundary. Python binding uses one-shot heap allocation + scs_heap_free; R uses size + fill in geometry.c.

4 · Voronoi (scs_voronoi_cells_2d_heap)

Euclidean Voronoi clipped to a bbox; single internal compute + heap copy.
