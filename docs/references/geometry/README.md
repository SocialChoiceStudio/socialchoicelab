# Geometry Services References

Exact and numeric geometric algorithms for SocialChoiceLab.

## Service Components

### geom2d
- CGAL EPEC exact 2D operations
- Exact predicates and constructions
- Robust geometric computations

### geom3d
- CGAL exact 3D operations where needed
- 3D convex hull algorithms
- 3D intersection computations

### geomND
- Numeric N-dimensional algorithms
- Convex hull in arbitrary dimensions
- Half-space intersection
- kd-tree for spatial queries
