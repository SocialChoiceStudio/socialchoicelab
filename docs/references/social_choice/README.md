# Social Choice Theory References

Theoretical foundations and practical implementations for social choice models.

## Core Concepts

### Preference Modeling
- Distance functions (Minkowski, Euclidean, Manhattan, Chebyshev)
- Loss functions (linear, quadratic)
- Utility function construction
- Indifference curve generation

### Voting Rules
- Plurality, Borda, Condorcet methods
- Approval voting
- Ranked choice voting
- Single transferable vote

### Aggregation Properties
- Transitivity
- Pareto efficiency
- Independence of Irrelevant Alternatives (IIA)
- Monotonicity
- Strategy-proofness

### Outcome Concepts
- Copeland score
- Strong Point
- Heart (uncovered set)
- Yolk (minimal covering circle)
- k-majority winsets
