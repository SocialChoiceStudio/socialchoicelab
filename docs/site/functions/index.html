    socialchoicelab.pairwise_matrix_2d(alt_xy, voter_ideals_xy, dist_config=None, k=-1) ¶

    Compute the n_alts × n_alts pairwise preference matrix.

    Entry [i, j] is: - 1 (SCS_PAIRWISE_WIN) if alt i is preferred by more voters than j. - 0 (SCS_PAIRWISE_TIE) if equal numbers prefer each. - -1 (SCS_PAIRWISE_LOSS) if alt j is preferred by more voters than i.

    The matrix is antisymmetric: M[i,j] = -M[j,i], M[i,i] = 0.

    Parameters:

    Name Type Description Default
    alt_xy

    Flat [x0, y0, …] alternative positions.

    		 required
    voter_ideals_xy

    Flat [x0, y0, …] voter ideal points.

    		 required
    dist_config DistanceConfig | None

    Distance configuration. None defaults to Euclidean.

    		 None
    k int

    Majority threshold (-1 = simple majority).

    		 -1

    Returns:

    Type Description
    numpy.ndarray of shape ``(n_alts, n_alts)`` and dtype ``int32``.
